# Project Context

## Project

LifePlus Pets marketing workspace focused on campaign planning, product messaging, creative continuity, and landing-page definition for the Spanish market.

## Current Reality

This workspace is not an application repo. It is a documentation-and-assets workspace containing:

- PDF product sheets in sibling project directories
- existing Ahiflower creative PDFs in `PETS MARKETING MATERIAL/`
- recovered campaign planning documents created during this session

## Operating Rules

- Keep source-of-truth planning inside `openspec/`
- Treat product availability and product documentation as separate facts
- Use only support language and non-medical claims
- Distinguish between locally verified product sheets and externally observed web catalog
- Prioritize a focused launch scope over presenting the entire technical catalog at once
