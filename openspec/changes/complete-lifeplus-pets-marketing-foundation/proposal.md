# Proposal: Complete LifePlus Pets Marketing Foundation

## Why

The workspace contains partial campaign outputs, verified product documentation, and Ahiflower creative references, but the planning state was fragmented across chat and standalone markdown files. OpenSpec should become the repository source of truth so the campaign can continue deterministically.

## What Changes

- create source-of-truth specs for catalog, campaign, and landing
- centralize project context in `openspec/project.md`
- define a focused phase-1 launch scope
- preserve Ahiflower creative continuity as the visual benchmark
- document the next implementation step for landing-page copy and asset completion

## Expected Outcome

Future work on product messaging, content planning, or landing-page buildout can proceed from `openspec/` instead of relying on chat history or scattered notes.
