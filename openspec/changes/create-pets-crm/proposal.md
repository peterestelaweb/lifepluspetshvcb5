# Proposal: Create PETS Outreach CRM

## Why

The campaign now needs a shared operational CRM to track every PETS-related contact and ensure consistent follow-up.

## What Changes

- add CRM source-of-truth spec
- create a local CRM web app inside `openspec/context/crm/`
- include template CSV with required fields
- define workflow-ready fields for response and next-step tracking

## Expected Outcome

The team can start entering contacts immediately and manage follow-up from one place while keeping campaign execution aligned.
