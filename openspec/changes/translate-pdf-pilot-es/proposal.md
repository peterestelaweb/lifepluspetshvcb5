# Proposal: Translate PDF Pilot to Spanish

## Why

The team needs a practical method to translate English PETS marketing PDFs into Spanish while preserving design and image composition.

## What Changes

- add source-of-truth spec for PDF translation workflow
- implement OCR and overlay tooling under `openspec/context/pdf_translation/tools/`
- create a pilot translation map for one representative PDF
- generate one translated pilot output PDF for visual validation

## Expected Outcome

A validated pilot and reusable tooling that can be applied to additional PDFs after approval.
